工具表 D｜客戶優先度矩陣

決定哪些客戶該加碼 / 降頻 / 放棄

2x2 矩陣：成交可能性 × 投入成本
象限 1（觀望型）：低可能性 + 低成本 → 降頻維護
象限 2（快速贏）：高可能性 + 低成本 → 全力加碼 ✓
象限 3（直接放）：低可能性 + 高成本 → 給最後通牒
象限 4（戰略型）：高可能性 + 高成本 → 用 3 問快篩判斷
💡 判斷基準：
高/低的分界點是 3.5 分
• 可能性 4-5 = 高 ｜ 1-3 = 低
• 成本 4-5 = 高 ｜ 1-3 = 低

評分標準

分數 成交可能性 投入成本
1 分 態度曖昧，沒需求 1-2 次溝通就能推進
2 分 有興趣但不清晰 3-5 次溝通 + 少量調研
3 分 需求明確，態度中立 多次會議 + 深度提案
4 分 需求確認，態度正面 大量客製化 + 多部門
5 分 強烈需求，態度積極 極高溝通成本

• 可能性 1-3 + 成本 4-5 → 🔴 象限 3【直接放】→ 給最後通牒

🔵 象限 4 專用｜3 問快篩法

如果你的客戶落在「象限 4」（高可能性 + 高成本），用這 3 個問題快速判斷：

問題 1：合約金額夠大嗎？

✅ 任一個打勾 → 全力投入（這是大單，動員 team 支援，1個月內推進到簽約）

問題 2：這是戰略客戶嗎？

✅ 任一個打勾 → 戰略投入（長期價值高，但設時間上限：最多投入 3 週，沒進展就暫停）

問題 3：有人可以幫你分攤成本嗎？

✅ 有 2 個以上打勾 → 可以做（成本可控，但設時間上限：最多 2-3 週）
❌ 只有 1 個或沒有 → 降低優先級（改用標準方案，不客製化。客戶不接受就放棄）

💡 快速判斷流程：
1. 先看問題 1（大單？）→ 有勾就全力投入
2. 問題 1 沒勾 → 看問題 2（戰略客戶？）→ 有勾就戰略投入
3. 問題 1、2 都沒勾 → 看問題 3（有人幫忙？）→ 有 2 個勾就可以做
4. 三個問題都沒達標 → 改用標準方案或放棄
