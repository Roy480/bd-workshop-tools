# BD 工作坊工具表

這是「BD 成交方法論」工作坊的互動式工具表網頁。

## 包含的工具表

- **工具表 A** - 個人漏斗診斷
- **工具表 B** - 陌開訊息起草框架
- **工具表 C** - 推進計畫表（2週版）
- **工具表 D** - 客戶優先度矩陣

## 功能特色

- ✅ 自動儲存（LocalStorage）
- ✅ 即時預覽（工具表 B）
- ✅ 自動計算（工具表 D）
- ✅ 響應式設計（手機/平板/電腦）
- ✅ 列印功能

## 部署到 Vercel

1. 將整個資料夾上傳到 GitHub
2. 在 Vercel 中 Import Project
3. 選擇這個 Repository
4. 點擊 Deploy

或者直接在 Vercel 上傳這個資料夾。

## 本地開發

直接用瀏覽器開啟 `index.html` 即可。

## 技術棧

- 純 HTML + CSS + JavaScript
- 無需任何框架或後端
- 資料儲存在瀏覽器 LocalStorage
